# 🩺 MediPulse

A full-stack healthcare platform built to connect **patients, doctors, and NGOs** through real-time communication, appointment management, and community engagement.

🔗 **Live Demo:** https://medi-pulse-gamma.vercel.app/

---

## ✨ Features

• Secure Authentication & Role-Based Access  
• Doctor Discovery & Profile Management  
• Appointment Booking Workflow  
• Real-Time Chat using Socket.IO  
• 1:1 Video Consultation using WebRTC  
• Community Posts & Events  
• NGO Integration & Engagement  
• Google Maps Location Support  
• Responsive Modern UI  

---

## 🛠 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Vite

### Backend
- Node.js
- Express.js
- MongoDB

### Realtime
- Socket.IO
- WebRTC

### Deployment
- Vercel

## 📂 Project Setup

```bash
# Frontend
cd frontend
npm install
npm run dev

# Backend
cd backend
npm install
npm start
```

Create `.env`

```env
MONGODB_URI=
JWT_SECRET=
GOOGLE_MAPS_API_KEY=
```
