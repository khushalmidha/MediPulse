# MediPulse

MediPulse is a full-stack healthcare platform that connects patients, doctors, and health communities in one simple web experience. Users can create accounts, discover doctors, join support communities, chat in real time, view community events, book appointments, and join live consultation rooms once a doctor starts the session. Appointment booking includes email OTP verification, a virtual wallet balance, automatic booking fee debit, email-based user approval or cancellation, queue-based doctor approval, refunds, appointment history, and downloadable medical receipts. Doctors can manage their queue, start and end appointments, add notes, generate receipts, create communities, and publish events for members.

The dashboard highlights recommendations, specialized doctors, and upcoming events from the communities a user has joined. The profile popover shows the user name, email, initials avatar, role, and current wallet balance. MediPulse also includes AI-assisted recommendations and chat support for a more guided healthcare journey.

The frontend is built with React, Vite, React Router, Tailwind CSS, Axios, Socket.IO Client, Lucide React, jsPDF, and Google Identity Services. The backend uses Node.js, Express, MongoDB, Mongoose, JWT authentication, cookies, Socket.IO, Nodemailer, Redis-compatible caching and locks, Kafka-ready virtual payment events, Google Auth Library, Razorpay support, and Gemini AI integration. The project is structured for local development and deployment with environment-based configuration, Docker support, and separate frontend and backend services.
